# Decisions

## 2026-04-25

- Use `SharedPreferences` for the MVP because DataStore was not already available and no new dependencies should be installed without approval.
- Keep navigation as a small in-memory enum because the MVP has only four screens and adding Navigation Compose would add another dependency.
- Use `PackageManager` to list launchable apps and store selected package names locally.
- Include Usage Access permission explanation and settings shortcut, but do not implement usage reading or enforcement in this slice.
- Set `compileSdk` and `targetSdk` to 35 because no installed Android SDK was discoverable in this workspace to derive a local value from.
- Do not pin Gradle's Kotlin toolchain to JDK 17 because this machine exposes JDK 21; keep generated bytecode targeting JVM 17 instead.
- Keep usage limit comparison in a small plain Java class so a temporary dependency-free unit test can run without adding JUnit. Replace this with a standard Android/JUnit test setup when dependency policy allows.
- Use `UsageStatsManager.queryUsageStats` for today's selected-app foreground usage and keep enforcement out of scope.
